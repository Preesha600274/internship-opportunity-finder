# 🎯 Internship Opportunity Finder

AI-powered agent that finds real internship opportunities using 
web search and LLM reasoning.

## Tech Stack
- Streamlit, Python
- Groq LLaMA 3.1 8B
- Tavily Search API
- Streamlit Cloud (Deployment)

## Agents
1. Search Agent — Tavily web search
2. Formatter Agent — LLM formats results  
3. Judge Agent — LLM evaluates output

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. Add .env with API keys
4. streamlit run app.py
